Juniper路由器开启snmp以及启用Netflow采集配置指南

1.进入配置模式命令edit

2.配置snmp命令:set snmp community juniper authorization read-write

3.查看软件版本show version

4.配置sflow

admin# set protocols sflow collector x.x.x.x udp-port 6343

admin# set protocols sflow interfaces ge-0/0/0 polling-interval 30 sample-rate egress 1000

admin# set protocols sflow sample-rate egress 1000

admin# set protocols sflow sample-rate ingress 1000

admin@EX4200# run show sflow //查看全局配置

admin@EX4200# show protocols sflow //查看具体配置

admin@EX4200# commit //使能、保存配置
